## underscore简介

### 什么是underscore?

* 它是一个JavaScript库
* 它主要是提供了与数组有关的操作。但是也有一些其他常用的操作。

```
var _=require("underscore");
console.log(_.functions(_));
```
// ['_','after'....'zip']
